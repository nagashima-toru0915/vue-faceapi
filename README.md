# vue-faceapi

使用方法
node.jsを使用します。
node.jsをインストールしてください。


## vue.jsをインストール。バージョン３を入れてください。
```
npm install -g @vue/cli
```

## gitから当ファイルをclone。

## face-api.jsを追加。
```
npm install @tensorflow/tfjs-node
npm install face-api.js
npm install canvas
```


### 実行する場合
```
npm run serve
```

### Electronでアプリ化
```
npm run electron:build
```
